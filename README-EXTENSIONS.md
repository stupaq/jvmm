Implemented language extensions
===============================

+   arrays
+   structures
+   objects
+   virtual methods
+   reference counting GC

External libraries used in the project
======================================

+   `bnfc`
+   `pretty-show`
+   `pandoc`
+   `llvm-general-pure` and `llvm-general` which provide AST for LLVM IR and utilities to write it into a `.bc` module.
